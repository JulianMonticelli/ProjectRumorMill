# ProjectRumorMill
Project Rumor Mill

To run a simulation:
   Change config import to appropriate python config
   python simengine.py

   Happy simulating!

We use:
   Python 2.7
   networkx v1.11 (Simulation)
   pytest
   Django (graph viewer)
   Matplotlib + pyplot
	-> (for one simulation graph display - nothing special)
   
To run all tests in main directory:
   pytest ./
